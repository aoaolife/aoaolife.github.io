硬件篇

  • MOTO G 2014，2015年8月份买的，没有想到这么耐用(cao)，各种刷机折腾都没有问题，现在仍然是主力机。
  • Lenove E431（已停产），其实我也想用Mac,然而这是公司配的。
  • Apple iPad mini 4，本来买回来是给老妈看视频聊微信用的，结果现在我每次回家总是占着它。
  • 吉列手动剃须刀锋速3，配合剃须膏，手动刮胡子也是一种享受了。
  • 罗马仕 充电宝，10000 豪安刚刚好，用了也两年多了吧。
  • New Balance 574 ，从14年开始穿，作为基本款还是很不错的。

软件篇

通用

  • 翻墙服务 自购ss账户,搬瓦工vps备胎，改善浏览网络体验

windows 7

编辑器和 Terminal

  • Sublime Text 3，用了好几年了，轻量，好用，主题是 Agila

其他相关插件：

  • HTML-CSS-JS Prettify 有了它，前端代码洁癖们可以安心了。
  • Babel 有了它，ES6的代码高亮也不愁了。

  • markdown editing 没错，有了它，我就是在用st3编辑markdown。

  • GitHub Pages，程序员记录必备，博客静态生成器使用 Hexo，个人博客地址：http://hktkdy.com/

系统相关

  • ccleaner ，一路相伴，哈哈哈。
  • everything，据说是很多人至今不愿意放弃windows的一个理由
  • 迅雷极速版，偶尔下个片也不错
  • IDM，接管浏览器下载功能

效率

  • Xshell ，服务器命令行工具
  • WinSCP ，服务器文件可视化操作工具
  • Axure RP Pro 7.0，原型图绘制工具
  • 福昕阅读器，阅读PDF足矣
  • Dropbox，文件同步工具，已经很久不用国内云服务做文件同步了，信不过。

音视频

  • potplayer,用了两三年了，只能说经典
  • 网易云音乐，免费

浏览器

  • Firefox,曾经多年的主力浏览器，现在很少用了。
  • Chrome，现在的主力浏览器，日常工作都很给力。

Chrome 插件

  • search-engine-filter，搜索结果过滤，我自己写的东西当然要放到第一位说。
  • uBlock Origin，过滤烦人的广告
  • Advanced REST client，Chrome 下接口数据调试
  • IDM Integration Module，接管浏览器自带下载功能
  • Evernote Web Clipper，印象笔记剪藏
  • Infinity新标签页，第三方新标签页，看了下，好几个同事也在用
  • LastPass: Free Password Manage，密码管理工具，也用了好几年了。
  • Markdown Here，偶尔会用，能够在富文本编辑器中使用markdown语法。
  • Proxy SwitchyOmega，翻墙必备
  • Save to Pocket，也是日常生活中比较常用的Chrome 插件之一了，配合安卓版pocket碎片化阅读也很不错。
  • qiniu upload files ，写博客上传图片，再也不用打开七牛官网琐碎地操作了。
  • React Developer Tools，调试React 项目必备
  • Stylish ,给任意网站添加任何你想要的样式
  • Tampermonkey ，给任意网站添加脚本支持，让浏览器从此与众不同
  • 右键搜 ，让你的右键更加个性化
  • 网页截图:注释&批注 ,顾名思义

油猴脚本

  • bilibili ASS Danmaku Downloader 下载B站视频弹幕，使用频度不高，但一旦用上，就是利器。
  • direct google ，去除google 重定向
  • old-smzdm(旧版张大妈) ，打开smzdm.com 直接重定向到旧版本，我自己的作品
  • 谷歌翻译提示框扩展 ，在阅读英文时，辅助以google翻译。
