亿纬锂能（300014.SZ）深度投资价值研究报告：2026年业绩拐点与全球化布局的戴维斯双击

报告日期： 2026年2月13日

投资评级： 买入（Buy）/ 强力增持

风险等级： 中高风险（R4）

1. 核心投资逻辑与决策建议

站在2026年2月13日这一时间节点，对于“今日是否购买亿纬锂能股票”这一问题，本报告基于详尽的基本面分析、财务预测模型及行业趋势研判，给出明确的肯定回答。亿纬锂能（EVE Energy）正处于一个极为关键的经营拐点：从2024-2025年的资本开支高峰期与股权激励费用摊销期，正式迈入2026年的产能释放期与利润兑现期。

1.1 核心推荐理由

首先，业绩剪刀差的出现是当前买入的最强逻辑。2025年公司业绩因高额的股权激励费用摊销及特定客户（如合众汽车）的坏账计提而受到压制，呈现出“增收不增利”的表象。然而，进入2026年，随着股权激励费用的显著下降以及前期投入的巨大产能（特别是荆门大圆柱工厂和马来西亚储能工厂）开始贡献规模化收入，公司净利润预计将实现报复性反弹。市场一致预期2026年归母净利润有望突破73亿元人民币，较2025年同比增长接近60%，这种内生性的利润修复将直接驱动估值中枢的上移 。

其次，大圆柱电池（46系列）的技术红利期已经到来。作为宝马集团“新世代”车型（Neue Klasse）的核心供应商，亿纬锂能在46系列大圆柱电池上的布局早于大多数竞争对手。2026年是宝马新世代车型的量产前夜，也是亿纬锂能大圆柱产线从“爬坡”转向“满产”的关键年份。公司在荆门、沈阳及匈牙利的产能布局构筑了极深的护城河，使其在下一代电池技术竞争中占据了有利身位 。

第三，“CLS”全球合作模式的有效性验证。面对美国《通胀削减法案》（IRA）带来的地缘政治挑战，亿纬锂能独创的“合作（Cooperation）、授权（License）、服务（Service）”模式（即CLS模式）已在美国ACT项目中落地。2026年将是该模式开始贡献技术授权收入的元年，这不仅规避了重资产出海的风险，更将公司的商业模式从单纯的制造业延伸至高毛利的技术服务业 。

1.2 交易时机研判

从二级市场交易层面看，2026年2月13日是一个具备极高性价比的击球点。当前股价（约64.19元）对应2025年业绩的市盈率（PE）约为29倍，看似不低，但若切换至2026年预期业绩，其动态市盈率已回落至18倍左右。考虑到公司未来三年超过30%的复合增长率（CAGR），其PEG（市盈率相对盈利增长比率）远小于1，处于显著低估区域。此外，困扰公司的证券虚假陈述索赔案件诉讼时效将于2026年4月18日届满，这一悬在头顶的利空靴子即将落地，市场风险偏好有望在随后显著回升 。

综上所述，建议投资者在当前位置积极配置亿纬锂能，以捕捉2026年业绩爆发与估值修复带来的“戴维斯双击”机会。

3. 财务深度透视：穿越周期的韧性与爆发力

亿纬锂能的财务报表是理解其投资价值的核心密码。我们需要穿透2025年的“财务迷雾”，看清2026年的真实盈利能力。

3.1 2025年业绩复盘：压力测试下的高质量增长

回顾2025年，亿纬锂能交出了一份“营收创新高，利润受压制”的答卷。前三季度，公司实现营业收入450.02亿元，同比增长32.17%；但归母净利润为28.16亿元，同比下滑11.70% 。这一看似矛盾的数据背后，实则隐藏着公司经营质量的显著提升：

  1. 营收含金量高：在锂价大幅波动、产业链价格战激烈的背景下，营收逆势增长30%以上，说明公司的市场份额（Market Share）在持续扩大，客户粘性极强。特别是在储能与动力电池双赛道上，出货量增速均超过35%，远超行业平均水平 。
  2. 一次性因素的干扰：2025年的净利润下滑主要受两点非经营性因素影响：
    • 股权激励费用：全年预计确认约11亿元的股权激励费用。这是一项非现金支出，虽然拉低了报表利润，但却是为了绑定核心人才，为长远发展蓄力 。
    • 坏账计提：上半年对合众汽车（哪吒汽车）的应收账款进行了1.5亿元的全额坏账计提。这一“利空出尽”的操作消除了后续资产负债表的不确定性 。
  3. 剔除后的真实盈利：若剔除上述股权激励费用与坏账计提的影响，2025年前三季度的归母净利润实际上达到了36.75亿元，同比增长18.40%。这表明公司的主营业务盈利能力不仅没有下降，反而在逆境中保持了稳健增长 。

2026年利润爆发的三大驱动力：

  1. 费用端的“减负”：2026年股权激励费用的摊销金额将大幅减少。这一会计处理上的变化将直接释放出数亿元的账面利润，成为业绩增长的“基石”。
  2. 大圆柱电池的量产效应：亿纬锂能是全球46系列大圆柱电池的领跑者。2026年，荆门大圆柱工厂将进入满产状态，沈阳工厂产能也将陆续释放。大圆柱电池因其高技术壁垒，享有远高于方形电池的毛利率（预计可达20%以上）。随着高毛利产品占比的提升，公司整体盈利结构将得到质的飞跃 。
  3. 储能业务的规模效应：随着马来西亚工厂的投产以及“Mr. Big”等大容量电芯的全球交付，储能业务将从单纯的“冲量”转向“量利齐升”。特别是面向海外市场（如美国、欧洲、澳洲）的高溢价订单，将显著增厚公司利润 。

4. 业务基石与第二增长曲线：双轮驱动战略

亿纬锂能之所以能在激烈的行业洗牌中屹立不倒，归功于其均衡且极具前瞻性的业务布局：以消费电池为基石，以动力电池为矛，以储能电池为盾。

4.1 动力电池：大圆柱技术的王者归来

在动力电池领域，亿纬锂能没有选择在方形电池的红海中与宁德时代进行低效内卷，而是另辟蹊径，押注46系列大圆柱电池。这一战略选择在2026年被证明是极具远见的。

技术壁垒与竞争优势： 46系列大圆柱电池（如4680、4695）具备高能量密度、高安全性及快充性能优势，是下一代高端电动汽车（尤其是800V平台）的首选方案。亿纬锂能在此领域的技术积累深厚，良品率处于行业领先水平。根据弗若斯特沙利文的报告，2024年亿纬锂能在全球大圆柱电池市场份额已位居第二，仅次于特斯拉自产 。

客户版图的扩张：

2026年是亿纬锂能大圆柱电池的“交付大年”。

  • 宝马集团：作为宝马新世代车型的定点供应商，亿纬锂能正在匈牙利建设大规模产能以满足宝马的需求。这一合作不仅带来了巨额订单，更背书了公司的技术实力 。
  • 造车新势力：在国内，公司已成功切入小鹏汽车（P7+、G6）、零跑汽车（C10、C16）等主流车企的供应链。2025年内已新增2家乘用车客户，配套车型数量达10款。随着这些爆款车型的放量，亿纬锂能的装机量将持续攀升 。

产能释放节奏： 公司透露，大圆柱电池新产品将在2025年三季度开始量产交付，预计到2026年，荆门大圆柱电池工厂将达到满产状态。此外，沈阳工厂的第二条大圆柱产线也在同步规划建设中。这种产能投放节奏与下游客户的需求爆发点完美契合，最大程度地避免了产能闲置风险 。

4.2 储能电池：全球化布局的先锋

储能业务已成为亿纬锂能增长最快的一极，2025年前三季度储能电池出货量高达48.41GWh，同比增长35.51%，营收占比大幅提升至近40% 。

产品力的代际领先： 亿纬锂能率先推出了628Ah超大容量储能电芯“Mr. Big”以及5MWh储能系统“Mr. Giant”。这两款产品通过极简集成设计，大幅降低了全生命周期成本（LCOE），在海外市场极具竞争力。2026年，这两款拳头产品将在全球范围内大规模铺开 。

海外市场的深度渗透：

  • 美国市场：尽管面临关税壁垒，但美国储能需求强劲（主要来自AIDC及电网侧）。亿纬锂能通过CLS模式及马来西亚工厂的转口贸易，成功绕开了部分贸易限制，继续分享美国市场的高增长红利 。
  • 澳洲市场：公司与澳大利亚能源企业EVO Power签署了2.2GWh的战略合作协议，这标志着公司在澳洲表前储能市场取得了重大突破。产品已获得AS 3000、AS 5139等当地准入认证 。

5. 全球化战略：构建地缘政治下的安全供应链

在逆全球化与贸易保护主义抬头的背景下，亿纬锂能展现出了高超的战略智慧，通过“CLS模式”与“海外建厂”双管齐下，构建了一个安全、灵活且高效的全球供应链体系。

5.1 马来西亚工厂：东南亚的战略支点

2026年Q1，亿纬锂能位于马来西亚吉打州的生产基地将全面建成投产 。该项目总投资不超过86.54亿元，主要生产新型储能电池及消费类圆柱电池 。

战略意义深度解析：

  1. 规避关税壁垒：马来西亚作为中立的第三方国家，其出口产品在一定程度上可以规避美国及欧盟针对中国本土制造产品的直接关税制裁。这为亿纬锂能向欧美客户供货提供了一条“安全通道”。
  2. 辐射东南亚市场：东南亚地区正在经历能源转型的初期爆发阶段，对储能及电动两轮车电池的需求激增。马来西亚工厂将作为桥头堡，直接覆盖这一新兴增量市场 。
  3. 提升交付效率：相比于从中国国内发货，马来西亚工厂能更快速地响应全球客户（特别是澳洲和南亚客户）的交付需求，缩短物流周期，降低运输成本 。

5.2 匈牙利工厂：欧洲本土化的关键一步

亿纬锂能在匈牙利德布勒森建设的电池工厂，是其全球化版图中最为重要的一块拼图。该工厂毗邻宝马集团的匈牙利工厂，旨在为宝马“新世代”车型提供近距离配套服务 。

应对“电池护照”： 欧盟《新电池法》对电池的碳足迹、回收比例等提出了严苛要求（即“电池护照”）。通过在欧洲本土建厂，亿纬锂能可以利用当地的绿电资源生产，显著降低产品的碳排放，从而满足欧盟法规的合规性要求，确保在欧洲市场的长期竞争力 。

5.3 CLS模式：轻资产出海的典范

针对美国市场《通胀削减法案》（IRA）对电池组件产地的限制，亿纬锂能创新性地推出了“CLS全球合作经营模式”。

  • C (Cooperation)：与当地合作伙伴建立合资关系。
  • L (License)：向合资公司进行技术授权。
  • S (Service)：提供工程建设及运营管理服务。

在这一模式下，亿纬锂能无需投入巨额资本建设重资产工厂，而是通过技术输出和管理输出来获取稳定的专利费和服务费。这不仅规避了地缘政治风险，还将公司的收入结构从单一的产品销售升级为“产品+服务+技术”的多元组合。2026年，随着美国ACT合资项目的推进，CLS模式将开始贡献实质性的利润 。

6. 行业宏观环境：供需再平衡与锂价企稳

投资个股离不开对行业大势的判断。2026年的锂电行业，正在经历从“内卷”到“出清”，再到“再平衡”的关键转折。

6.1 碳酸锂价格的“新常态”

经历过2023-2024年的暴跌后，碳酸锂价格在2026年2月已企稳在13.8万-14.2万元/吨的区间 。这一价格水平对于产业链而言是“舒适区”：

  • 对上游：该价格已击穿部分高成本云母矿和非洲矿的成本线，导致落后产能被迫出清（如宜春地区的环保整顿和采矿权注销），供给侧收缩明显 。
  • 对下游：相比于高峰期的60万元/吨，当前的锂价显著降低了电池成本，使得电动车和储能系统更具性价比，从而刺激终端需求的释放。

对于亿纬锂能而言，锂价的稳定意味着库存减值风险的彻底消除。公司可以更加从容地进行原材料采购和库存管理，毛利率的波动性将大幅降低。

6.2 供需格局的改善

中金公司研报指出，2026年锂电行业将迎来“底部反转”。虽然电池环节整体产能仍有富余，但在高端产能（如大圆柱、快充电池、长循环储能电芯）方面，供需关系趋于紧张 。 亿纬锂能聚焦于高端差异化产品，其产能利用率（稼动率）预计在2026年将维持在高位。特别是在储能领域，随着全球AIDC（人工智能数据中心）对电力需求的激增，储能配建需求有望超预期爆发，进一步消化行业产能 。

7. 风险因素的全景扫描与压力测试

尽管投资逻辑强硬，但作为严谨的投资者，我们必须审视潜在的风险点，并评估其对公司价值的冲击。

7.1 证券虚假陈述索赔风险（即将解除）

7.2 港股IPO的摊薄风险

风险描述：亿纬锂能已于2026年1月再次向港交所递交IPO申请，计划发行H股募集资金用于匈牙利工厂建设 。 影响评估：H股发行确实会在短期内对A股股东权益造成一定摊薄。然而，从中长期看，这是利大于弊的战略举措：

  1. 资金支持：匈牙利工厂是资本密集型项目，通过港股融资可以降低公司的资产负债率（目前约63.5%），优化资本结构 。
  2. 国际化形象：港股上市有助于提升公司的国际品牌形象，便利海外客户的合作与互信。
  3. 融资渠道：打通境外融资平台，为未来的全球化并购或扩张提供便利。

7.3 地缘政治与贸易摩擦

风险描述：欧美国家可能针对中国电池企业出台更严厉的关税或供应链限制政策。

对冲策略：亿纬锂能已通过马来西亚建厂和美国CLS模式做好了充分的风险对冲。即使发生极端的贸易封锁，公司依然保留了通过第三方国家或技术授权方式服务欧美市场的能力。
