大学时代。C语言老汪的课。
汪：“今天出勤率挺可怜的啊。这样啊，在座的同学们，我给你们出一道题，满分20分记到期末总分里。没来的后来的咱们不管他们。他们没听见题自己倒霉。先别写啊，等我出下一道题的时候你们一块儿写。都听好了，用C语言给我在屏幕上写两坨大便，什么样儿的都行。两坨。”
我：“老师，能不能重复一遍，或者把题写下来？”
汪：“我就说一遍！那么几个字儿还用往黑板上写吗？你们寝室是不是有人没来？想诈我，你小子挺鬼啊。呵呵。”
我心想，呵呵你妹，我管他们去死。你自己出的题有歧义，好心提醒你tm这么对我……
过了一小会儿，好心室友们call来不少人。
汪：“同学们，今天出勤率不错，我给你们出一道题，满分5分记到期末总分里，就当点名了。用C语言在屏幕上给我打一首春晓。”
于是我刷刷刷写下了这样的程序：

```
#include <stdio.h>
#include <stdlib.h>
void main()
{
system("CLS");
(void)printf( "liangtuodabian,shenmeyangerdedouxing.\nliangtuo.\nyishouchunxiao.\n" );
}
```

从落笔写到交卷，一共没用上两分钟。还检查了一下自己的汉语拼音有没有问题。
还沾沾自喜，我竟然还记得M$记的系统窗口上怎么清屏，此处应该有加分。
老汪看我第一个交卷，顺手就拿过去批。
他的表情包非常丰富，最后还是忍不住破口大骂：“你脑子有病啊，你这能画出来两坨大便啊！打这么几个字用你啊！你觉得一个printf能值20分啊！”
我：“老师，你刚才说的是‘写’不是‘画’。我提醒你重复一遍，你又不听。”
汪：“以后你不用来上我课了，期末我直接给你90分。出去！”

看，我的脑回路即使在休眠状态也是这么卓尔不凡。
